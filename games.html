Doodle Jump

A vertical platformer where players control a bouncing character, guiding it upward by landing on platforms while avoiding obstacles.

Flappy Bird

A side-scrolling game where players tap to keep a bird airborne, navigating it through gaps between green pipes without crashing.

Sudoku

A number-placement puzzle where players fill a 9×9 grid with digits 1-9, ensuring no repetition in rows, columns, or 3×3 subgrids.

Wordle

A word puzzle game where players guess a five-letter word in six tries, receiving color-coded hints for correct letters and positions.

Minesweeper

A logic-based puzzle game where players uncover tiles on a grid, avoiding hidden mines while using numerical hints to deduce safe spaces.

Snake

A classic arcade game where players control a growing snake, guiding it to eat food while avoiding collisions with itself and the screen edges.

Candy Crush

Candy Crush Saga is a colorful match-three puzzle game where players swap candies to complete challenges and progress through sugary levels.

Crossy Road

Crossy Road is an endless arcade hopper where players guide a character across busy roads, rivers, and train tracks while avoiding obstacles and aiming for the highest score.

Connect Four

Connect Four is a two-player strategy game where players drop colored discs into a vertical grid, aiming to connect four in a row before their opponent.

Tic Tac Toe

Tic-Tac-Toe is a classic two-player strategy game where players take turns marking Xs and Os on a 3×3 grid, aiming to get three in a row.

Memory Game

Memory Game is a card-matching challenge that tests players' concentration by flipping pairs of hidden cards to find matching images or symbols.

Guess the Number

Number Guessing Game is a simple logic-based game where players try to guess a hidden number within a given range, receiving hints after each attempt.
